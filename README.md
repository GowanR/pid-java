# pid-java
A quick and easy java implementation of PID.
Used for FIRST Robotics Competition. Highly experimental. *Made for educational purposes.*
Now with ramping setpoints and integral-windup-prof integrals!

## How to Test.

Grab yourself a copy of [junit and hamcrest.](https://github.com/junit-team/junit4/wiki/Download-and-Install)

Run `sh run_tests.sh` in your terminal.
